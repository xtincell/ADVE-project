The 8 pillars of ADVERTIS (the acronym = the execution order):
- **A** — Authenticité (identity, vision, myth) — first, feeds all
- **D** — Distinction (positioning, visual identity, voice) — puise dans A
- **V** — Valeur (value proposition, pricing, economics, business model) — puise dans A+D
- **E** — Engagement (devotion ladder, touchpoints, rituals, superfan) — puise dans A+D+V
- **R** — Risk (SWOT, risk matrix, mitigations, overton blockers) — puise dans ADVE
- **T** — Track (market reality, triangulation, TAM/SAM/SOM, overton position) — puise dans ADVE+R
- **I** — Innovation (total brand potential, everything the brand CAN do) — puise dans ADVE+R+T
- **S** — Strategy (roadmap that PICKS FROM I to move Overton Window → superfan) — puise dans ADVE+R+T+I

**CRITICAL — Cascade order: A → D → V → E → R → T → I → S = ADVERTIS**
The execution order IS the acronym. Each pillar consumes all previous ones.

**I = Innovation, NOT Implementation.** I is the brand's total potential — everything it can be, do, become.
**S = Strategy, NOT Synthèse.** S is the roadmap that PICKS from I's catalogue to build the action plan.
**S comes AFTER I because S pioche dans I.** You can't choose (S) before knowing what's possible (I).

**North Star: the SUPERFAN.** Every pillar serves the same goal — moving the Overton Window to accumulate superfans (evangélistes in the Devotion Ladder).
