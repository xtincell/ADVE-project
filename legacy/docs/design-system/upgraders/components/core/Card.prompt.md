Bento surface primitive — tile these in a CSS grid to build dashboards. Variant sets the treatment, `interactive` adds hover-lift.

```jsx
<Card>Default dark cell</Card>
<Card variant="raised" interactive>Elevated, lifts on hover</Card>
<Card variant="glass">Translucent overlay panel</Card>
<Card variant="accent" padding="8">Prêt à passer au niveau supérieur ?</Card>
<Card variant="light">White editorial block on a dark page</Card>
```

Variants: `default · raised · glass · accent · light · inkmax`. Props: `padding` (0/4/5/6/8), `radius` (lg/xl/2xl), `interactive`.
