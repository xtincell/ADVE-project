Circular avatar (image or initials) + overlapping `AvatarGroup` with "+N" overflow — for client lists, message threads, "+1,2k entrepreneurs".

```jsx
<Avatar src={url} name="John D." status="online" />
<Avatar name="Awa Diop" size="lg" ring />
<AvatarGroup max={4} avatars={[{src:a},{src:b},{src:c},{name:"Kadi S."},{src:e}]} />
```

Sizes `xs·sm·md·lg·xl` or a number. Status `online·busy·offline`.
