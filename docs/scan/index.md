## Project Overview

- **Type**: monolith (single Next.js app)
- **Primary Language**: TypeScript 6 (strict)
- **Architecture**: 6-layer strict hierarchy (`domain → lib → server/governance → server/services → server/trpc → components/neteru → app`), governance cascade via `mestor.emitIntent`, hash-chained Intent log, NSP SSE streaming, multi-tenant default-deny
